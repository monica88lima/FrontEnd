HyperText Mark-up Language - HTML

HTML é a sigla em inglês para HyperText Markup Language, que significa Linguagem de Marcação de Hipertexto em português. É uma linguagem de marcação usada para criar páginas da web.

O que é HTML?

O HTML permite que os desenvolvedores estruturem o conteúdo de uma página web usando elementos como cabeçalhos, parágrafos, imagens e links, entre outros. Os elementos HTML são compostos de tags (etiquetas) que são inseridas no código-fonte de uma página da web para indicar como o conteúdo deve ser apresentado no navegador. Essas tags são interpretadas pelo navegador para exibir o conteúdo da página da forma como o desenvolvedor deseja. Em resumo, o HTML é uma linguagem fundamental para a criação de páginas da web, pois é responsável por definir a estrutura e o conteúdo da página.

Tags

As tags HTML são elementos fundamentais da linguagem HTML, que são usadas para marcar e estruturar o conteúdo de uma página da web. As tags são compostas por uma palavra-chave delimitada por colchetes angulares, como:
<html>,<head>, <body>, <p>,<img>, <a>, entre outras.
Cada tag HTML tem um propósito específico, que pode ser aplicado a diferentes tipos de conteúdo, como texto, imagem, vídeo, formulários e links.
Por exemplo, a tag <p>é usada para definir um parágrafo de texto, enquanto a tag <img> é usada para inserir imagens na página.
Existem várias tags HTML que são usadas para exibir texto na página da web. Aqui estão alguns exemplos de tags HTML de texto:

  • <p> - é a tag usada para criar um parágrafo de texto.
  • <h1> a <h6> - são as tags usadas para criar títulos e subtítulos em ordem hierárquica, sendo o <h1> o mais importante e o <h6> o menos importante.
  • <strong> - é a tag usada para enfatizar um texto em negrito.
  • <i> - é a tag usada para enfatizar um texto em itálico.
  • <u> - é a tag usada para sublinhar um texto.
  • <blockquote>- é a tag usada para exibir citações em bloco.
Essas são apenas algumas das tags HTML mais comuns usadas para formatar e exibir texto na página da web. É importante lembrar que o uso correto das tags HTML é fundamental para garantir que o conteúdo seja exibido corretamente e que a página da web seja acessível para todos os usuários.
As tags HTML são interpretadas pelo navegador da web para renderizar a página de acordo com as instruções fornecidas pelo desenvolvedor. É importante lembrar que as tags HTML precisam ser usadas corretamente e fechadas adequadamente para garantir que a página seja exibida corretamente.

Atributos das Tags

Os atributos HTML são informações adicionais que são adicionadas às tags HTML para modificar o comportamento ou aparência de um elemento na página da web. Cada tag HTML pode ter diferentes atributos, que podem ser usados para especificar coisas como o tamanho ou a cor de um elemento, a URL de um link, a fonte de uma imagem, entre outras coisas.
Os atributos são definidos dentro da tag HTML, após o nome da tag e antes do fechamento do colchete angular. Por exemplo, a tag <a> pode ter um atributo href que especifica a URL do link, como:
<a href = "https://www.google.com">.
A tag <img> pode ter um atributo src que especifica o caminho para a imagem, como:
<img src="imagem.jpg" alt="Descrição da imagem">.
Os atributos também podem ser usados para criar interatividade na página da web, como no caso dos formulários, que utilizam atributos como name, value e type para permitir que o usuário insira e envie dados para o servidor.

É importante lembrar que os atributos HTML devem ser usados corretamente e de acordo com as especificações da linguagem, para garantir a acessibilidade, a compatibilidade e a funcionalidade da página da web.
